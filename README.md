# digital_attendence
Simple PHP + MySQL digital attendance system (XAMPP). Shows login, entry form, and attendance view.

## Features
- Teacher login
- Add attendance entries
- View all attendance records

## Tech Stack
- PHP (mysqli)
- MySQL (via XAMPP)
- HTML/CSS
